# WooCommerce Staff POS

Staff-facing assisted checkout for WooCommerce with a dedicated POS cart, customer lookup, and payment link/manual payment order flows.

## Plugin Metadata

- Name: `WooCommerce Staff POS`
- Author: `Shakeel Ahmad`
- Version: `0.1.0`
- Text domain: `wc-staff-pos`
- REST namespace: `wc-pos/v1`

## Current Scope

- Staff-only WooCommerce admin page
- Dedicated POS cart isolated from the storefront cart
- Simple and variable products
- Customer search and creation
- Pending orders with WooCommerce payment links
- Manual paid orders for in-store checkout
